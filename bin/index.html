Front-End

El primer paso para dominar cualquier habilidad es entender los fundamentos de la misma. En este primer tema aprenderas lo basico sobre HTML, CSS, JavaScript y otras herramientas fundamentales en la construccion de sitios web.

HTML

Hyper Text Markup Language o HTML es el lenguaje que se encarga de crear la estructura del sitio web. Este lenguaje compone los cimientos sobre los que se construira tu sitio web.

CSS

Cascade Style Sheets o CSS se encarga de definir el estilo que seguiran los elementos de tu web. A traves de este lenguaje dotaras a los elementos de tu HTML de un diseño atractivo.

JavaScript

JavaScript es un lenguaje que se encarga de dotar a tus sitios web de interactividad, cuando haces click en un boton, abres un menu desplegable u observas cambios automaticos en una pagina web, estas siendo testigo del uso de Javascript.

WebKits

Una parte fundamental en cualquier tipo de desarrollo es el uso de herramientas que potencien tanto la eficiencia como la optimizacion del resultado final. En el desarrollo front-end nos encontramos con diversos frameworks y herramientas que haran que tanto el proceso de creacion como el resultado final sean lo mas optimos y eficientes posible.

Layouts

Una de las claves para que un diseño sea atractivo, es la disposicion que siguen los elementos que lo componen, es decir su Layout. En este segundo tema aprenderas diversas tecnicas para la creacion de Layouts, desde el uso de display block o inline hasta la tecnica de mas reciente auge conocida como flex-box.

Posicionamiento

Una primera aproximacion al diseño de layouts es el uso tanto de posiciones absolutas como relativas. Esta primera opcion cuenta con limitaciones pero es un buen punto de partida en el aprendizaje sobre Layouts.

Floats

La propiedad float de CSS nos permite posicionar elementos de forma que aparezcan uno al lado del otro y aunque esto puede producir ciertas limitaciones, es una opcion mas que viable.

Grid Layout

El diseño en Grid es un diseño que permite el manejo bidimensional del Layout, a traves de el aprenderas a controlar la disposicion de elementos mediante el uso de un container.

Flex Layout

El mas reciente diseño denominado flex-box layout nos permita una mayor flexibilidad a la hora de determinar la posicion de elementos y aunque cuenta con la limitacion de ser unidimensional, es una de las mejores opciones para la construccion de diseños.

UI/UX

No todo en el desarrollo front-end es programacion, un aspecto fundamental en el desarrollo de sitios web es el diseño de la interfaz de usuario y el manejo de la experiencia de usuario. En este tercer tema aprenderas los fundamentos que componen un buen diseño de UI/UX para que tus sitios web ofrezcan la mejor experiencia posible.

UI

La interfaz de usuario es el canal a traves se comunicara tu sitio web con los usuarios por lo que es fundamental adoptar un diseño que sea lo mas atractivo y usable posible.

UX

Diseñar una buena experiencia de usuario es clave para conseguir una buena retencion de audiencia; usabilidad, interactividad y adaptabilidad son solo tres factores que engloban el campo de la UX.

Usabilidad

Es necesario que tu sitio web hable el mismo lenguaje que tus usuarios para que estos no tengan ningun problema al hacer uso del mismo. Esto es posible gracias a la usabilidad, dotando tus sitios web de facilidad de uso y aprendizaje.

Protocolos

Existen ciertas tecnicas y protocolos que se han demostrado en numerosas ocasiones que son efectivos a la hora de realizar el diseño de UI/UX. Siempre es importante dotarlos de personalidad pero es buena idea empezar con una base que ya se ha comprobado anteriormente como efectivas.
